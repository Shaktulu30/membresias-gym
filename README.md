# Simulador de Membresias GYM

Pagina web interactiva que simula la contratación de planes de membresía
de un gimnasio. El usuario elige un plan y una duración, y la app calcula el
precio final, aplicando descuentos según el caso.

## Funciones
- Visualización de los planes disponibles con sus precios y beneficios
- Selección de duración (1, 6 o 12 meses)
- Cálculo automático del precio final
- Descuento del 15% al contratar 12 meses
- Notificaciones al contratar un plan

## Herramientas
- JavaScript (manipulación del DOM, eventos, fetch)
- HTML y CSS
- Toastify (librería de notificaciones)
- Datos cargados desde un archivo JSON
